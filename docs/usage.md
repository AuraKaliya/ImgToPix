# 使用手册

## 启动 GUI

Windows 下直接双击：

```text
start.bat
```

或命令行启动：

```bash
python bootstrap.py gui
```

首次启动会自动：

- 创建 `.venv`
- 安装项目依赖
- 启动图形界面

## 基本流程

1. 导入或拖拽一张图片
2. 选择补边模式
3. 选择处理预设
4. 查看 `16x16`、`32x32`、`64x64` 预览
5. 选择输出目录
6. 点击“保存全部”

## 输出文件

默认会生成：

- `原文件名_16.png`
- `原文件名_32.png`
- `原文件名_64.png`

若勾选“同时保存放大预览图”，还会额外生成：

- `原文件名_16_preview.png`
- `原文件名_32_preview.png`
- `原文件名_64_preview.png`

## 常见场景

- 把伪像素立绘压缩成头像尺寸
- 为游戏素材准备固定规格像素图
- 批量把一组风格统一的图片导出为多个像素规格
