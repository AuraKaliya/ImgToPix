# 使用手册

## 启动 GUI

Windows 下可以直接双击：

```text
start.bat
```

或命令行启动：

```bash
python bootstrap.py gui
```

首次启动会自动：

- 创建 `.venv`
- 安装项目依赖
- 启动图形界面

## 基本流程

1. 导入或拖拽一张图片
2. 选择补边模式
3. 对比多个 `16x16` 母版算法
4. 观察当前选中方案的 `16x16`、`32x32`、`64x64` 预览
5. 选择输出目录
6. 点击“保存全部”

## 输出文件

默认会生成：

- `原文件名_16.png`
- `原文件名_32.png`
- `原文件名_64.png`

其中：

- `16x16` 是实际母版
- `32x32` 与 `64x64` 由母版最近邻放大得到

若勾选“同时保存放大预览图”，还会额外生成：

- `原文件名_16_preview.png`
- `原文件名_32_preview.png`
- `原文件名_64_preview.png`

## 适用场景

- 将放大过的伪像素图还原为小尺寸 sprite
- 为游戏素材准备统一规格的像素输出
- 批量比较不同 `16x16` 还原算法的效果
