# GUI 说明

## 设计目标

GUI 面向不想记命令的用户，强调：

- 一次导入即可生成三种尺寸
- 改参数就能实时看到结果
- 输出路径和参数可记忆
- 处理过程不阻塞界面

## 界面结构

- 左侧显示原图与补边结果
- 右侧显示 `16x16`、`32x32`、`64x64` 三种输出
- 顶部区域提供导入、重新生成、保存、输出目录和参数选择

## 参数说明

- `边缘延展`：将边缘像素向外延伸，最适合当前项目默认目标
- `镜像补边`：对边缘内容做镜像反射，适合内容靠边但希望过渡更自然的图片
- `纯色补边`：使用纯色补齐，适合背景明确的素材

- `标准`：平衡细节与稳定性
- `更锐利`：边缘更强，适合轮廓比较明确的图
- `更平滑`：减少噪点，适合原图有压缩痕迹时使用
