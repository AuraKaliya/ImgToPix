# ImageToPixel

把高分辨率“伪像素”图片压缩为真正的像素图，并固定输出 `16x16`、`32x32`、`64x64` 三种规格。

## 功能

- 导入常见图片格式
- 非 `1:1` 图片自动补成正方形
- 支持边缘延展、镜像补边、纯色补边
- 支持标准、更锐利、更平滑三种处理预设
- 同时预览原图、补边图和三种输出尺寸
- 一键导出 PNG
- 可选导出放大预览图
- 记住上次使用的目录和参数

## 安装

```bash
pip install -r requirements.txt
```

## 运行

```bash
python app.py
```

## 测试

```bash
pytest
```
